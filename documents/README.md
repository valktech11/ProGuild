Session handover documents — pick up cleanly in a new chat.
- `HANDOVER_v106.md` — latest (June 7, 2026)
- `HANDOVER_v106.docx` — same, formatted

### /plans
Build plans and strategy documents.
- `ProGuild_Roofing_Master_Plan.docx` — roofing CRM build plan (updated June 7, 2026)
- `ProGuild_Founders_Bible_Addendum_v5_9_Jun7.docx` — §24 Twilio/SMS, §25 Session fixes (append to Bible v5_9)

- `ROOFING_CRM_current_state.md` — what is actually built in the repo vs. what docs say (June 7, 2026)

## How to use
1. Start each session: share HANDOVER_vXXX.md as context
2. End each session: update ROOFING_CRM_current_state.md + write new HANDOVER
3. Bible and Roadmap are the deep reference — update addendums, not the core docs
